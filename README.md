# 🧮 Calculadora Java

Aplicação de linha de comando desenvolvida em Java que realiza operações aritméticas básicas entre dois números.

---

## 📋 Funcionalidades

- Soma (`+`)
- Subtração (`-`)
- Multiplicação (`*`)
- Divisão (`/`) — com validação: impede divisão por zero ou por número negativo

---

## 🚀 Como executar

### Pré-requisitos

- [Java JDK](https://www.oracle.com/java/technologies/downloads/) 8 ou superior instalado
- Terminal / Prompt de comando

### Compilar

```bash
javac Main.java
```

### Executar

```bash
java Main
```

---

## 🖥️ Exemplo de uso

```
Digite o primeiro número: 10
Digite o segundo número: 2
Digite o simbolo para o calculo (+), (-), (/), (*)
/
Resultado = 5.0
```

---

## ⚙️ Operadores disponíveis

| Símbolo | Operação       |
|---------|----------------|
| `+`     | Soma           |
| `-`     | Subtração      |
| `*`     | Multiplicação  |
| `/`     | Divisão        |
